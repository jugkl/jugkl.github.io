In Kooperation zwischen Chaos in KL, JUG-KL und Softwerkskammer RLP findet der Global Day of Code Retreat dieses Jahr erstmals auch in Kaiserslautern statt. Ein Code Retreat bedeutet Rückzug vom üblichen Projektgeschehen und Konzentration auf elementare Programmierelemente wie Clean Code, TDD oder Design. Eine Aufgabenstellung wird im Pair Programming mit wechselnden Partnern und unter verschiedenen Bedingungen wieder und wieder bearbeitet. Die Herangehensweise und Programmiersprache ist dabei frei wählbar.

Der Global Day of Code Retreat findet bereits zum achten Mal statt. Erwartet werden dieses Jahr über 200 Events weltweit. Mehr Infos dazu unter https://www.coderetreat.org/

Die Teilnahme ist kostenlos, aber eine Anmeldung ist erforderlich. Bitte meldet euch auf der Seite der Softwerkskammer an, sodass wir besser planen können: http://softwerkskammer-rlp.de/events/4/global-day-of-coderetreat/

17.11.2018, 10:00 - 17:00, Chaos Computer Club Kaiserslautern
